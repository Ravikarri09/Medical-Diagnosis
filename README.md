# Deep Learning Based Medical Diagnoser (RNN)

Built an LSTM-based deep learning model to predict diseases and recommend
medications from patient symptom descriptions.

## Key Features
- Text preprocessing using tokenization and padding
- LSTM-based sequence learning
- Multi-output classification (Disease + Prescription)
- Saved model and encoders for inference
- Optional Streamlit-based deployment

## Tech Stack
Python, TensorFlow, Keras, NumPy, Pandas, Scikit-learn
